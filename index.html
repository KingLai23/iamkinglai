Some Of My Projects.

Puzzle - A Minecraft Server Manager

A simple server manager for the popular game, Minecraft. Built for the Linux terminal, it allows you to start, stop, backup, restore, and set automated backups for your Minecraft server. Given a number of maximum backups, it will auto-remove the oldest backups to make sure you never exceed the maximum amount. This saves tons of storage space each month! Everything is backed onto an SVN repository.

Dotted - Puzzle Game

A fun puzzle game whose goal is to collect as many points as possible. Swap two dots to form chains of length 3 or more to earn points. Every chain created is removed from the game board, where the pieces cascade downwards, and the remaining empty slots are filled with randomly colored pieces. Dotted utilizes recursion to produce the cascade, and feautures a simple and attractive GUI.
